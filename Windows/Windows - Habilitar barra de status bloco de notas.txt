Digite regedit no campo de busca do menu Iniciar e navegue até HKEY_CURRENT_USER\Software\Microsoft\Notepad. No painel direito, localize a chave StatusBar, clique nela com o botão direito do mouse e altere o número do campo Dados do Valor de 0 para 1. Clique em OK para finalizar e feche o Editor do Registro. Pronto. Se você abrir o Notepad verá que a barra de status já está habilitada.
